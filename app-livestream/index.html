1.LiveStream#

유튜브나 페이스북을 이용한 실시간 동영상을 전송하도록 지원한다.

2.H264 프로그램 설치#

pi> cd /usr/src  
pi> git clone git://git.videolan.org/x264  
pi> cd x264  
pi> ./configure --host=arm-unknown-linux-gnueabi --enable-static --disable-opencl`  
pi> make  
pi> sudo make install

3.FFMpeg 프로그램 설치#

pi> cd /usr/src
pi> git clone https://github.com/FFmpeg/FFmpeg.git  
pi> cd ffmpeg  
pi> sudo ./configure --arch=armel --target-os=linux --enable-gpl --enable-libx264 --enable-nonfree
pi> make   
pi> sudo make install

4.패이스북 또는 유튜브 계정 등록#

계정등록 후 해당 URL을 획득한다. [SERVER URL]/[STREAM NAME]

5. 전송프로그램 실행#

pi> raspivid -o - -t 0 -vf -hf -fps 30 -b 6000000 | ffmpeg -re -ar 44100 -ac 2 -acodec pcm_s16le -f s16le -ac 2 -i /dev/zero -f h264 -i - -vcodec copy -acodec aac -ab 128k -g 50 -strict experimental -f flv [SERVER URL]/[STREAM NAME]
