메일 보내기#

  1. Register a Twitter App with this tutorial to get Consumer Key etc.
  2. Install Python and required libraries with following commands
  3. sudo apt-get install python-setuptools
  4. sudo easy_install pip

  5. sudo pip install smtplib

  6. Download source code Link

  7. wget https://raw.githubusercontent.com/sixfab/rpiShields/master/tutorials/tutorial7/SecuritySystem.py

  8. Make it executable

  9. sudo chmod +x SecuritySystem.py

  10. Run it

  11. sudo python SecuritySystem.py
