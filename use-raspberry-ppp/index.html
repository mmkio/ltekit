1.LTE Module 연결#

  • USB LTE Module SIM 카드 장착
  • USB LTE 모듈 과 라즈베리파이 연결
  • 라즈베리파이 전원, 모니터, 키보드, 마우스 케이블 연결.

simcard 삽입

2.USB 연결상태 확인#

USB LTE 모듈 과 라즈베리파이 연결 (USB 연결) 및 확인

pi> cd ~       #--> 홈디렉토리 이동
pi> ls /dev

ttyUSB1, ttyUSB2, ttyUSB3 의 연결 상태를 확인한다.

3.pppd 프로그램 설치 및 실행#

pi> sudo apt install ppp                 #--> pppd 프로그램 설치

4.ppp 스크립트 파일을 실행#

스크립드를 wget 을 이용하여 다운로드 한다.

pi> wget https://raw.githubusercontent.com/mmkio/ltekit/master/doc/pppd-ec25-make.sh


pi> sudo chmod +x ./pppd-ec25-make.sh
pi> sudo ./pppd-ec25-make.sh lte-internet.sktelecom.com  #--> PN:'lte-internet.sktelecom.com'

Note

  • "lte-internet.sktelecom.com"은 통신사 접속 APN을 나타내며 사용되는 통신 SIM 카드에 따라 변경된다.
  • APN 은 SIM 카드와 함께 제공된다.

국내 주요 APN 목록

  • KT : lte.ktfwing.com
  • SKT : lte.sktelecom.com, lte-internet.sktelecom.com, ltetab.sktelecom.com
  • LG U+ : internet.lguplus.co.kr

5.인터넷 접속 디바이스 확인#

pi> ifconfig

6.인터넷 접속 디바이스 확인#

현재 사용되고 있는 이더넷 연결 포트는 eth0, wlan0, wwan0 으로 포시되고 있으며, 현재 사용되고 있는 포트를 사용 중지한다. lo 는 내부에서 사용되는 포트 임으로 사용중지 대상에서 제외한다.

pi> sudo ifconfig eth0 down
pi> sudo ifconfig wlan0 down
pi> sudo ifconfig wwan0 donw


pi> ifconfig

7.PPP 데몬 실행 및 인터넷 연결#

pi> sudo pppd call lte

8.통신연결상태확인#

새로운 터미널 창을 하나 생성하여 인터넷 통신 테스트 확인을 한다.

pi> ifconfig
pi> ping 8.8.8.8          #구글 DNS 서버 주소
pi> ping www.google.com   #구글 웹사이트 주소
